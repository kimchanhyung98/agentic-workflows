# 로컬 개발 자동화 에이전트

> AI CLI로 기획 문서를 작성 → 사람이 리뷰 → 기획 기준으로 자율 실행

## 개요

Stripe Minions의 아키텍처를 로컬 환경에 맞게 변형한 개발 자동화 에이전트이다.
AI CLI(Claude Code 등)를 에이전트 코어로 활용하고, 실행 전 사람의 리뷰를 거치는 구조를 취한다.
실행 환경은 로컬과 VM/Docker 모두 지원한다.

## 핵심 원칙

1. **하네스를 설계하라** — 에이전트가 아닌 시스템 환경(hook, 게이트, 기획 문서)에 투자
2. **LLM + Gate 분리** — 코드 생성은 LLM에게, 검증은 결정론적 게이트에
3. **기존 도구 활용** — 에이전트 전용 도구를 만들지 않고 AI CLI, lint, test, Git을 활용
4. **제한된 재시도** — 자동 수정 횟수를 제한하고, 초과 시 사람에게 에스컬레이션
